Dewey Analytics
Nonfiction circulation by Dewey 100-range & 3-digit class
Updated Quarterly or As Needed: Queries 5E (ANONFIC, JNONFIC, YANONFIC) on the Collection Analysis page generate the borrower breadth data shown in each class sub-table.
Metabase Queries — Dewey Class Depth
Borrower breadth analysis at the 3-digit Dewey class level for Adult Nonfiction. Reveals whether a class's circulation is driven by broad community demand or repeat-patron inflation. Classes with fewer than 5 items are excluded. Run in Metabase, export as CSV, and share with Claude to update the dashboard.
SELECT
    CAST(LEFT(REGEXP_SUBSTR(i.itemcallnumber, '[0-9]+'), 3) AS UNSIGNED) AS dewey_class,
    COUNT(DISTINCT i.itemnumber)                         AS total_items,
    ROUND(AVG(COALESCE(i.issues, 0)), 2)                AS avg_lifetime_circ,
    SUM(COALESCE(i.issues, 0))                           AS total_circs,
    COUNT(DISTINCT s.borrowernumber)                     AS unique_borrowers,
    COUNT(DISTINCT CASE WHEN i.issues > 0
          THEN i.itemnumber END)                         AS items_circulated,
    ROUND(COUNT(DISTINCT CASE WHEN i.issues > 0
          THEN i.itemnumber END)
          / COUNT(DISTINCT i.itemnumber) * 100, 1)      AS pct_items_circulated,
    ROUND(COUNT(DISTINCT s.borrowernumber)
          / NULLIF(SUM(COALESCE(i.issues, 0)), 0), 3)   AS borrower_breadth
FROM items i
LEFT JOIN statistics s
    ON  s.itemnumber = i.itemnumber
    AND s.type = 'issue'
WHERE i.homebranch              = 'BERTWP'
  AND i.ccode                   = 'ANONFIC'
  AND i.itemlost                = 0
  AND i.withdrawn               = 0
  AND COALESCE(i.notforloan, 0) = 0
  AND i.itemcallnumber REGEXP '[0-9]'
GROUP BY CAST(LEFT(REGEXP_SUBSTR(i.itemcallnumber, '[0-9]+'), 3) AS UNSIGNED)
HAVING total_items >= 5
ORDER BY dewey_class;

Collection Analysis
Weed candidates, DoA risk items, high-use titles, & duplicates
Updated Quarterly or As Needed: Queries 5A–5D below generate the working item lists in the tabs below.
Metabase Queries — Collection Analysis
Item-level list of weed candidates. Criterion A: no checkout in 3+ years (all collections, 4-yr protection window). Criterion B: nonfiction with copyright 15+ years and fewer than 3 lifetime checkouts. Excludes on-loan, lost, withdrawn. Limit 500, sorted oldest-last-circ first.
SELECT
    b.title,
    b.author,
    i.itemcallnumber     AS call_number,
    i.ccode              AS collection,
    i.datelastborrowed   AS last_circ,
    i.barcode
FROM items i
JOIN biblio b ON b.biblionumber = i.biblionumber
LEFT JOIN biblioitems bi ON bi.biblionumber = i.biblionumber
WHERE i.homebranch       = 'BERTWP'
  AND i.itemlost         = 0
  AND i.withdrawn        = 0
  AND i.notforloan       = 0
  AND i.onloan           IS NULL
  AND i.dateaccessioned  < DATE_SUB(NOW(), INTERVAL 4 YEAR)
  AND (
        (i.datelastborrowed IS NULL
         OR i.datelastborrowed < DATE_SUB(NOW(), INTERVAL 3 YEAR))
        OR
        (i.ccode IN ('ANONFIC', 'JNONFIC', 'YANONFIC')
         AND bi.publicationyear IS NOT NULL
         AND bi.publicationyear < YEAR(NOW()) - 15
         AND COALESCE(i.issues, 0) < 3)
      )
ORDER BY i.datelastborrowed ASC
LIMIT 500;
Items accessioned 2–6 months ago with zero checkouts. New items that aren’t moving — candidates for relocation, display promotion, or early review. Excludes Library of Things, Periodicals. Limit 300.
SELECT
    b.title,
    b.author,
    i.itemcallnumber   AS call_number,
    i.ccode            AS collection,
    i.dateaccessioned  AS date_added,
    i.barcode
FROM items i
JOIN biblio b ON b.biblionumber = i.biblionumber
LEFT JOIN statistics s
    ON  s.itemnumber = i.itemnumber
    AND s.type       = 'issue'
WHERE i.homebranch = 'BERTWP'
  AND i.itemlost   = 0
  AND i.withdrawn  = 0
  AND i.ccode NOT IN ('LIBOFTHNGS', 'APERIOD', 'JPERIOD')
  AND i.dateaccessioned BETWEEN DATE_SUB(NOW(), INTERVAL 6 MONTH)
                             AND DATE_SUB(NOW(), INTERVAL 2 MONTH)
GROUP BY
    i.itemnumber, b.title, b.author,
    i.itemcallnumber, i.ccode, i.dateaccessioned, i.barcode
HAVING COUNT(s.datetime) = 0
ORDER BY i.dateaccessioned ASC
LIMIT 300;
Items with 20+ lifetime checkouts in the statistics table. High-demand titles that may need replacement copies or repair review. Threshold set at 20 (not 50) to account for statistics table history limits. Excludes Library of Things. Limit 500.
SELECT
    b.title,
    b.author,
    i.itemcallnumber    AS call_number,
    i.ccode             AS collection,
    COUNT(s.itemnumber) AS lifetime_checkouts,
    i.datelastborrowed  AS last_circ,
    i.barcode
FROM items i
JOIN biblio b ON b.biblionumber = i.biblionumber
JOIN statistics s
    ON  s.itemnumber = i.itemnumber
    AND s.type       = 'issue'
WHERE i.homebranch = 'BERTWP'
  AND i.itemlost   = 0
  AND i.withdrawn  = 0
  AND i.ccode     != 'LIBOFTHNGS'
GROUP BY
    i.itemnumber, b.title, b.author,
    i.itemcallnumber, i.ccode, i.barcode, i.datelastborrowed
HAVING COUNT(s.itemnumber) >= 20
ORDER BY lifetime_checkouts DESC
LIMIT 500;
Bib-level view of titles with 2+ copies in the same collection. Includes Copies and On Loan columns to judge whether duplicates are justified by demand. Excludes Library of Things, Periodicals, Chinese collection. Limit 500.
SELECT
    b.title,
    b.author,
    MIN(i.itemcallnumber)                                  AS call_number,
    i.ccode                                                AS collection,
    COUNT(i.itemnumber)                                    AS copies,
    SUM(CASE WHEN i.onloan IS NOT NULL THEN 1 ELSE 0 END) AS on_loan
FROM items i
JOIN biblio b ON b.biblionumber = i.biblionumber
WHERE i.homebranch = 'BERTWP'
  AND i.itemlost   = 0
  AND i.withdrawn  = 0
  AND i.notforloan = 0
  AND i.ccode NOT IN ('LIBOFTHNGS', 'APERIOD', 'JPERIOD')
  AND i.itemcallnumber NOT LIKE 'CHINESE%'
GROUP BY i.biblionumber, b.title, b.author, i.ccode
HAVING COUNT(i.itemnumber) > 1
ORDER BY copies DESC, b.title ASC
LIMIT 500;
